# Persistence with JPA

## General info
A minimalist persistence project. Persists a user and messages in an SQLite database.
	
## Technologies
Project is created with:
* Gradle 6.3
* Ipsum version: 2.33
* Ament library version: 999
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
